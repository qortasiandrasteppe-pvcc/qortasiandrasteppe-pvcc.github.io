    Operating Systems Terminology

Term Definition
Operating System Software that controls the overall operation of a computer
Job Queue The place where jobs reside in mass storage to wait for execution
FIFO First-In, First-Out
Interactive processing Operating system that allowed a program being executed to carry on a dialogue with user through remote terminals
Time-sharing Operating system that provided service to multiple users at the same time
Multi Programming Technique of time sharing where time is divided into intervals then the execution of each job is restricted to only one interval at a time
Multi Programming Technique of time sharing where time is divided into intervals then the execution of each job is restricted to only one interval at a time
Application Software Consists of the programs for performing tasks particular to the machine's utilization
System software Performs tasks that are common to computer systems in general
Utility Software Software units that extend the capabilities of the operating system
Graphical User Interface Objects to be manipulated, such as files and programs, are represented pictorially on the display as icons
Shell Older user interface that communicated with users through textual messages using a keyboard and monitor screen
Kernel Internal part of an operating system
Device Driver Software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine
File Manager Coordinate the use the machine's mass storage facilities
Memory Manager Coordinates the use of the machine's use of main memory
Virtual Memory When main memory shuffles pages back and forth between main memory and mass storage so that the pages are needed at any given time are actually present in the 4GB of main memory
Boot Strapping Transers the operating system from mass stoarage into main memory
ROM A small portion of a computer's main memory where the CPU expects to find its initial program is constructed from special nonvolatile memory cells. Such memory is known as read only memory
Deadlock Condition in which two or more processes are blocked from progressing because each is eating for a resource that is allocated to another
Spooling Technique of holding data for output at a later but more convenient time
Super User A person that gains highly priviledged access to the operating system by identifying themselves as the administrator
Multifactor Authentication A security method that provides more than one piece of evidence to gain access
