    Operating Systems Terminology

Term Definition
LAN Local Area Network
MAN Metropolitan Area Network
WAN Wide Area Network
Bus Network Communication Line
Star Network Communication line popular in wireless networks where communication is conducted by means of radio broadcast
CSMA/CD Ethernet Protocol dictates that each message be broadcast to all the machines on the bus
CSMA/CA Wifi Wireless network that adopts policy of trying to avoid collisions, rather than detecting them
Repeater Device that passes signals back and forth between the two original buses
Bridge Connects to buses, but does not pass all messages across the connection. Instead it looks at the destination and only forwards the message when it is meant for a computer on the other side
Switch A bridge with multiple connections, connect several buses
Router Special purpose computer used for forwarding messages
Gateway Point at which one network is linked to an internet
P2P: Peer-to-Peer Processes that provide service to and receive service from another
Distributed System Software units that execute as processes on different computers
Cloud Computing Huge pool of computers on the network can be allocated for use by clients as needed
Tier 1 Very high speed, high capacity, international WANs. Backbone of internet
Tier 2 Regional in scope and less potent in their capabilities
Tier 3 Most or all of the network remains connected even when very important tier 1 routers fail
ICANN Non-profit corporation established to coordinate the internet's operation
DNS Domain Name System, internet wide directory system
TLD: Top-Level-Domain Suffix following period
Country-Code TLDs Two letter TDLs for each specific country
FTP Application for accessing another pomputer from a great distance
HTTP Hyper Text Transfer Protocol
SMTP A way two computers on the network may interact when transmitting an email message from one host to another
URL A document with a unique address called uniform resource locator
Client-side Activities Activities performed by client
Server-Side Activities Activities performed by server
Application Layer Message originates here
Transport Layer Passes down through the transport
Network Layer Prepares for transmission
Link Layer Transmitted by link layer
Packet Switching Network Digital data is sent by breaking the data into blocks of bits called packets
Fault Tolerant Internet is tolerant of many types of faults
Redundant The internet can fall back to other redundant pathways for delivery when congestion or failure occur
Malware Malicious software
Virus Software that infects a computer by inserting itself into programs that already reside in the machine
Worm Autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
Trojan Horse Program that enters a computer system disguised as a desirable program
Spyware Software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
Phishing Method of obtaining information explicitly by simply asking for it
DDoS Attack Overloading a computer with messages
Firewall Primary prevention technique to filter traffic passing through a point in the network
Spam Filter A firewall designed to block unwanted email
Proxy Server Software unit that acts as an intermediary between client and server with the goal of shielding the client from adverse actions of the server
Antivrus Software Detect and remove the presence of known viruses and other infections
Cryptography Study of sending andn receiving messages securely in the presence of adversaries
Public Key Encryption Techniques by which the encryption and decryption algorithms are known
